# snowIRT
This module includes dichotomous, polytomous, and many facet model based on Rasch measurement model. This module also provides variouse types of DIF,and Distractor analysis.

# Installation
At the moment, snowIRT is not in CRAN yet, so you need to install it via devtools

devtools::install_github("hyunsooseol/snowIRT")
