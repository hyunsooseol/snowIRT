# snowIRT

`snowIRT` is a jamovi module developed and maintained by snowsoft Inc.

It is part of the snowsoft statistical module series for jamovi and provides Rasch measurement, item response theory, DIF, and distractor analysis tools for education, research, and applied data analysis.
